📊 Banking Dashboard Analysis (Power BI)

📌Project Description
This project presents an interactive Power BI dashboard developed to analyze banking data and provide meaningful insights into financial performance and customer trends. The dashboard focuses on key metrics such as total deposits, loans, account distribution, and customer segmentation.
It enables users to explore data through dynamic visualizations, filters, and slicers, helping identify patterns, monitor performance, and support data-driven decision-making in the banking sector.

🎯Objectives
Analyze overall banking performance
Track deposits and loan distribution
Understand customer segmentation
Identify trends in financial activities
Support strategic decision-making

📊 Key Features
KPI cards (Total Deposits, Loans, Customers)
Loan and deposit analysis
Customer segmentation insights
Time-based trend analysis
Interactive slicers and filters

📂 Dataset Information
The dataset includes:
Customer details
Account information
Loan and deposit records
Transaction or financial data
Time-based attributes

🛠️ Tools & Technologies
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Visualization Techniques

📈 Key Insights
Overview of total deposits and loans
Identification of high-value customers
Trends in banking transactions
Performance comparison across segments
